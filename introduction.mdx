## Overview

Vandar helps teams create, organize, and publish documentation efficiently. You manage all project docs in one centralized space with real-time collaboration, version control, and customizable themes. Whether you build APIs, user guides, or technical specs, Vandar streamlines your workflow.

Support multiple languages like English and Arabic (`فاندر`) out of the box. Customize layouts, add interactive components, and integrate with tools like GitHub or Slack.

## Brand Guidelines

<Expandable title="Primary Brand Color and Usage" default-open="true">

Use the primary brand color `#3B82F6` for headers and accents.

| Element       | Color Value   | Usage Example                  |
|---------------|---------------|--------------------------------|
| Primary       | `#3B82F6`     | Buttons, links, headers        |
| Secondary     | `#1D4ED8`     | Hover states, secondary UI     |
| Text          | `#1F2937`     | Body text, labels              |

Avoid using the primary color for backgrounds to maintain readability.

</Expandable>

<Expandable title="Logo Guidelines">

- Always use SVG format for scalability
- Minimum size: `120x40px`
- Clear space: 1x logo height around edges
- Do not stretch or recolor the logo

</Expandable>
