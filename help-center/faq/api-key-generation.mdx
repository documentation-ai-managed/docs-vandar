## Overview

To use Vandar APIs, you need an API key for authentication.

## Create Your API Key

1. Login to your Vandar dashboard.
2. Go to the **API** section.
3. Click **Create New Key**.
4. Copy and store your API key securely.

## Example Usage

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.vandar.io/v1/payments
```
